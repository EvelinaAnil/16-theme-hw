This repository contains C++ homework assignments and projects for "Theme 16". The projects are built and structured using Microsoft Visual Studio.

## Technologies Used

*   **Programming Language:** C++
*   **Environment / IDE:** Microsoft Visual Studio

---

## 📂 Repository Structure

The project is organized into a single workspace directory (`Theme16 hw`) containing two independent Visual Studio solutions. Each solution represents a distinct milestone or set of tasks within the homework scope.

```text
16-theme-hw/
├── Theme16 hw/
│   ├── 16 theme hw/                  # First Assignment Project Directory
│   │   ├── .vs/                      # VS local settings (should be ignored)
│   │   ├── 16 theme hw/              # Source files, headers, and assets
│   │   ├── x64/Debug/                # Compiled 64-bit binaries (Debug build)
│   │   └── 16 theme hw.sln           # Visual Studio Solution file
│   │
│   └── 16 theme hw 2/                # Second Assignment Project Directory
│       ├── .vs/                      # VS local settings (should be ignored)
│       ├── 16 theme hw 2/            # Source files, headers, and assets
│       ├── x64/Debug/                # Compiled 64-bit binaries (Debug build)
│       └── 16 theme hw 2.sln         # Visual Studio Solution file
---

## How to Run

1. Clone this repository to your local machine.
2. Navigate to the `Theme16 hw` directory.
3. Open either `16 theme hw.sln` or `16 theme hw 2.sln` using Microsoft Visual Studio.
4. Ensure your build configuration is set appropriately (e.g., `Debug` and `x64`).
5. Build and run the project using the "Local Windows Debugger" button in Visual Studio (or press `F5`).
